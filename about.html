Cryck

About

Cryck is a tool to help you visualize stats on CryptoHack and compare other users.

Features

  • See the amount of challenges needed to get a new trophy on each category
  • See challenges that you haven't done yet
  • Sort unsolved challenges by name, category, number of solves and number of points.
  • Compare two users: compare number of trophies and see which challenges were done by only one user.
